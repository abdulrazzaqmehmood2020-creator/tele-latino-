## ❓ Frequently Asked Questions

### Installation Questions
**Q: Do I need to install anything?**
A: No! Just download the files and open `calculator.html` in a browser. See [Installation Guide](./INSTALLATION.md#quick-start-3-steps).

**Q: What if I want to use a local server?**
A: We have instructions for VS Code, Python, and Node.js in [Installation Guide](./INSTALLATION.md#-for-developers-advanced-setup).

### Usage Questions
**Q: How do I use the calculator?**
A: Full instructions in [Usage Guide](./USAGE.md#basic-usage).

**Q: Can I use my keyboard?**
A: Yes! All keyboard shortcuts are in [Usage Guide](./USAGE.md#using-keyboard-faster).

**Q: Does it work on mobile?**
A: Yes! See [Usage Guide](./USAGE.md#-mobile-usage).

### Technical Questions
**Q: What technologies are used?**
A: HTML5, CSS3, and Vanilla JavaScript. See [Usage Guide](./USAGE.md#-learning-the-code).

**Q: Can I modify it?**
A: Yes! See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines.
