## 🤝 How to Use

### Calculator
1. Open `tools/calculator.html` or visit [the live demo](./tools/calculator.html)
2. Click number buttons to enter values
3. Select an operation (+, -, ×, ÷)
4. Click = to see the result
5. Click C to clear
